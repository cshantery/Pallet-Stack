# Pallet-Stack
Pallet Stack is a full stack Inventory Management web application for a Youngstown Pallet Shop
Users are able to manage inventory, orders, and invoices. 



## Features
    Inventory -
        Add
        Search
        Delete
        Update
    Orders-
        Add
        Search
        Delete
        Update
    Invoices-
        Add
        Search
        Delete
        Update

    In the future we would like to include shipment tracking, ability to export invoices, and more detailed inventory management


## Tech-Stack
    Backend - Python, Flask API
    Frontend- HTML, JS, CSS
    DataBase - MySQL server
    Python-DB-Connector - mysql-connector-python

## Dependacnies
    To run this application you must have:
        - MySQL server 
        - Python3
        - GIT
